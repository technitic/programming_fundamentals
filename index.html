What is a Programming Language?

a programming language is a set of instructions that enables humans to communicate with computers—using a series of symbols that serve as a bridge that allows humans to turn our ideas into instructions computers can understand. Like spoken languages, each language has its own unique syntax, structure, vocabulary, and even slang or shortcuts.

What is a Program?

A program is set of instructions that are written in a well defined structure to solve a particular problem. Program can be written in programming languages like C, C++ etc

1. Low level programming languages are the languages that are near to computer and far from human beings. These are understood by Computer easily and it is very difficult for humans to understand it

Low level Languages have two types

(a) Machine Language

(b) Assembly Language

Machine Language

Machine language is the most basic language a computer understands, consisting of binary code (0s and 1s) that directly controls the hardware. It’s extremely fast but hard for humans to write and understand

Assembly Language

Assembly language is a step above machine language, using simple, human-readable codes (called mnemonics) like MOV, ADD, and SUB to represent machine instructions. While it’s easier to work with than binary, assembly language still requires detailed knowledge of the computer’s architecture and needs to be converted into machine language by an assembler for execution.

2. High Level Language

A high-level language is a programming language that is designed to be easy for humans to read, write, and understand. Unlike machine language and assembly, which are closely tied to a computer's hardware, high-level languages use more natural language-like syntax and abstract away the complex details of hardware management. Examples of high-level languages include Python, Java, and C++
